
# Flash Chat ⚡️


## About the app

A messaging app where users can sign up and log in to chat.


##Main functionality

- Incorporate Firebase.
- Firebase authentication.
- Hero animations.
- Custom animations. 
- Using streams.
- Firebase Cloud Firestore.
